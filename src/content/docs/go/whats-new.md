## v0.1.0 — Phase 1

First release. Core framework + KurrentDB integration.

### Included

- Aggregate with fold pattern
- Functional command service (primary) and aggregate command service (optional)
- EventStore interfaces with LoadState/LoadAggregate/StoreAggregate
- TypeMap + JSON codec
- Subscription framework with middleware chain
- Checkpoint committer with gap detection
- KurrentDB store implementation
- KurrentDB catch-up and persistent subscriptions
- OpenTelemetry tracing and metrics
